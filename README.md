#	Number of Decks  

Polyglots and other polymaths tend to have dozens of decks. This little addon can help curb excesses. It adds a menu item to the <b>Tools</b> menu on the <b>Decks</b> screen (⌥⇧⌘D), which allows one easily to determine one’s total number of decks.
